# VeriCorp Templates

Starter templates for integrating the [VeriCorp API](https://rapidapi.com/vericorptestcollab/api/vericorp) into your projects.

## Templates

| Template | Stack | Description |
|----------|-------|-------------|
| [nextjs-vericorp](nextjs-vericorp/) | Next.js | Company verification form with server-side validation |
| [express-vericorp](express-vericorp/) | Express | REST API with company lookup and batch endpoints |
| [stripe-vericorp](stripe-vericorp/) | Express + Stripe | VAT validation before Stripe subscriptions |

## Get an API Key

Sign up at [RapidAPI](https://rapidapi.com/vericorptestcollab/api/vericorp) to get your key.
